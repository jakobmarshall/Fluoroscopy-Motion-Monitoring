# Fluoroscopy-Motion-Monitoring
Developed by Jakob Marshall & Steven Thomas at BC Cancer Vancouver

This application retrospectively monitors the position of radiopaque markers from bi-planar fluoroscopy images

This software is provided for research and educational purposes only. It is not intended for clinical use or any workflow that influences patient care. The authors and contributors make no warranties regarding accuracy, safety, or suitability for clinical decision‑making.
Use of this software is at your own risk.
